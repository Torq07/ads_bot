# Telegram Ad bot (reference: OLX in telegram)

## Features
* Item search
* Ability to add new sell item ( desctiprion, pictures, location)
* Local based search results
* Database logging

## Usage
Current bot is running at [@maasbot](https://telegram.me/maasbot).
